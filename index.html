O que é Segurança Digital?

É a proteção de informações digitais contra roubo, dano ou acesso indevido. Isso inclui desde dados pessoais até sistemas corporativos e governamentais.

Leis Brasileiras de Proteção

LGPD
Lei nº 13.709/2018

Regula o uso de dados pessoais e exige consentimento para coleta e uso de informações.

Marco Civil
Lei nº 12.965/2014

Estabelece direitos e deveres no uso da internet, garantindo privacidade e neutralidade.

Lei Carolina Dieckmann
Lei nº 12.737/2012

Tipifica crimes cibernéticos como invasão de dispositivos e roubo de dados.

Como se Proteger no Ambiente Digital

Senhas Fortes
  • Use combinações de letras maiúsculas, minúsculas, números e símbolos
  • Evite informações óbvias (datas, nomes)
  • Troque senhas regularmente
  • Use senhas diferentes para cada serviço
Autenticação de Dois Fatores (2FA)
  • Adicione uma camada extra de segurança
  • Use SMS, aplicativos autenticadores ou biometria
  • Ative em contas sensíveis
  • Protege mesmo se a senha for descoberta
Backup de Dados
  • Faça cópias regulares de arquivos importantes
  • Use nuvem segura ou HDs externos
  • Teste os backups periodicamente
  • Mantenha múltiplas cópias
Boas Práticas Gerais
  • Mantenha sistemas e aplicativos atualizados
  • Use antivírus e firewalls confiáveis
  • Não clique em links suspeitos
  • Desconfie de e-mails que pedem informações pessoais

Invasões Digitais: Principais Ameaças

Phishing

O que é: Golpe onde criminosos fingem ser instituições confiáveis para roubar dados como senhas e informações pessoais.

Como se proteger:

  • Verifique se o remetente é confiável
  • Nunca clique em links suspeitos
  • Ative a verificação em duas etapas
  • Use antivírus com filtro antiphishing
Malware

O que é: Software malicioso criado para danificar, espionar ou assumir controle de dispositivos. Inclui vírus, spyware e ransomware.

Como se proteger:

  • Mantenha o antivírus atualizado
  • Baixe apenas de sites confiáveis
  • Evite programas piratas
  • Não instale cracks
Worm (Verme Digital)

O que é: Tipo de malware que se espalha automaticamente entre computadores usando redes e dispositivos conectados.

Como se proteger:

  • Atualize o sistema operacional regularmente
  • Use firewall e antivírus eficazes
  • Desative funções de rede desnecessárias
  • Faça escaneamentos regulares
Trojan (Cavalo de Troia)

O que é: Malware que se disfarça de software inofensivo, mas abre caminho para invasores acessarem o sistema.

Como se proteger:

  • Instale apenas programas de fontes confiáveis
  • Evite sites não oficiais
  • Não use softwares piratas
  • Use soluções que detectem comportamento suspeito

Golpes Digitais Comuns

Golpes em Apps de Mensagem

Criminosos clonam números e fingem ser conhecidos para pedir dinheiro ou informações pessoais.

Como se proteger:
  • Ative verificação em duas etapas
  • Desconfie de pedidos urgentes
  • Nunca compartilhe códigos de verificação
  • Confirme a identidade por outros meios
Golpes em Compras Online

Sites falsos ou anúncios enganosos com preços muito baixos. Você paga mas não recebe o produto.

Como se proteger:
  • Verifique se o site é confiável
  • Pesquise avaliações de outros clientes
  • Use métodos de pagamento seguros
  • Evite clicar em links suspeitos
Golpes com Boletos Falsos

Boletos bancários adulterados que redirecionam o pagamento para contas de criminosos.

Como se proteger:
  • Gere boletos apenas em sites oficiais
  • Confira o nome do beneficiário
  • Verifique o código de barras
  • Use apps de bancos para escanear

Cyberbullying: Prevenção e Denúncia

O que é Cyberbullying?

É a prática de agredir, intimidar, humilhar ou assediar alguém por meio de plataformas digitais, como redes sociais, aplicativos de mensagens ou jogos online.

Exemplos de Cyberbullying
  • Disseminação de boatos e fofocas
  • Exposição de fotos ou vídeos íntimos sem consentimento
  • Criação de perfis falsos
  • Envio de mensagens ameaçadoras
  • Exclusão proposital de grupos online
  • Hacking de contas pessoais
Como Denunciar
  • Plataformas Digitais: Use ferramentas de denúncia das redes sociais
  • Disque 100: Serviço gratuito 24h para direitos humanos
  • Registro de Provas: Salve prints e evidências
  • Delegacia: Em casos graves, registre B.O.
  • Adultos de Confiança: Converse com pais e professores
Lei nº 14.811/2024

Esta lei incluiu o bullying e cyberbullying como crimes no Código Penal, com penas de multa e reclusão para intimidação sistemática por meios eletrônicos.

Boas Práticas de Navegação Segura

Seu Guia Rápido de Segurança
  • Use senhas fortes e diferentes
  • Ative autenticação de dois fatores
  • Não compartilhe dados em redes públicas
  • Pense antes de clicar em links
  • Mantenha softwares atualizados
  • Use conexões seguras
  • Cuidado com o que compartilha
  • Monitore suas contas regularmente
  • Use antivírus e firewall
  • Faça backups periódicos
